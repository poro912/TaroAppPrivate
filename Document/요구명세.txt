카드 내용 저장
카드의 긍정적인 의미와 부정적인 의미를  파일 또는 json 등의 파일기반시스템에 저장한다

저장 내용
카드 번호, 카드 제목, 정(긍정) 의미, 역(부정) 의미, 줄거리

파일의 내용을 읽어와 유동적으로 카드의 수를 파악하여 화면에 뿌려줘야함

스프레드 내용 저장
좌표 (x, y), rotate 순서번호 의미
ex) 100, 100, 0, 1, 과거

카드 선택기능
카드의 개수를 읽어와 화면에 뿌린다
화면에 뿌리는 방식은 미정으로 세가지가 있다
1. 카드 전체 보여주기(갤럭시타로)
2. 10장씩 보여주되 맘에들지 않다면 섞기 버튼을 눌러 다음으로 넘김(포스텔러)
3. 깔아놓고 스와프해서 이동(헬로우 봇)

결과화면
